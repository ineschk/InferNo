Fonctionnement

Simple à intégrer,
puissant à l'exécution

01
🌐
Connexion réseau
Une connexion entre dans votre réseau. L'agent extrait les 41 features NSL-KDD.
02
⚙️
Preprocessing
Sélection des 15 meilleures features, encodage et normalisation automatiques.
03
🤖
XGBoost prédit
Le modèle entraîné sur NSL-KDD calcule la probabilité d'attaque en <100ms.
04
🚨
Score retourné
Score 0–100 + label (normal / suspect / attaque) + niveau de confiance.
inferno-api — POST /v1/score
$ curl -X POST https://api.inferno.io/v1/score \
  -H "X-API-Key: ng_live_xxxx" \
  -d '{"features": [0,"tcp","http","SF",181,5450,...]}'

# Réponse en 47ms
{
  "score": 0,
  "label": "normal",
  "confidence": 1.0,
  "model_version": "1.0.0"
}

$ curl -X POST https://api.inferno.io/v1/score \
  -d '{"features": [0,"icmp","ecr_i","SF",1032,0,...]}'

# Attaque DoS détectée
{
  "score": 100,
  "label": "attaque",
  "confidence": 1.0
}
